# About

These are the docs for **smbCloud** — four services on one platform:
[Deploy](/deploy), [Auth](/auth), [Mail](/mail), and [GresIQ](/gresiq).

This site has two halves:

- **Platform** — product documentation for each service: what it does and how to
  use it.
- **[Developer](/developer)** — the technical reference for the CLI, tools, and
  SDKs you install and build against.

The Developer pages for the CLI are pulled from the
[smbcloud-cli](https://github.com/smbcloudXYZ/smbcloud-cli) repository, so they
stay in step with the tool itself.
