## How these docs stay current

The CLI and contributing pages here are **pulled from the
[`docs/`](https://github.com/smbcloudXYZ/smbcloud-cli/tree/main/docs) folder of
the [smbcloud-cli](https://github.com/smbcloudXYZ/smbcloud-cli) repository**
(`main` branch). They're synced into this site, so they track the tool itself —
edit them at the source, not here.
