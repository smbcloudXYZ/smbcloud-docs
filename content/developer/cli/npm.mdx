## Install

```bash
npm install -g @smbcloud/cli
```

Also works with pnpm, bun, and yarn:

```bash
pnpm add -g @smbcloud/cli
bun add -g @smbcloud/cli
yarn global add @smbcloud/cli
```

## Verify

```bash
smb --version
```

## Upgrade

```bash
npm update -g @smbcloud/cli
```

## Uninstall

```bash
npm uninstall -g @smbcloud/cli
```
