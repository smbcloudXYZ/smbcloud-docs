## Install

Run once without a persistent install:

```bash
uvx --from smbcloud-cli smb
```

Install as a persistent tool:

```bash
uv tool install smbcloud-cli
```

Or with pip:

```bash
pip install smbcloud-cli
```

## Upgrade

```bash
uv tool upgrade smbcloud-cli
```

Or with pip:

```bash
pip install --upgrade smbcloud-cli
```

## Uninstall

```bash
uv tool uninstall smbcloud-cli
```

Or with pip:

```bash
pip uninstall smbcloud-cli
```

---

All installation methods above use pre-built binaries — no Rust toolchain required. You can also download the binary directly from [GitHub Releases](https://github.com/smbcloudXYZ/smbcloud-cli/releases).
