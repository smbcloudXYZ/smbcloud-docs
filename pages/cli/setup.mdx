# Setup

Our CLI interface is build with Rust and published on [Crates.io](https://crates.io/crates/smbpndk-cli).

1. Setup your [Rust](https://www.rust-lang.org/tools/install).
2. Install it globally with cargo: `cargo install smbpndk-cli`.
3. Run `smb` command in your teminal to check your installation. You should get something like this:

```
╰─➤  smb
Command line tool for creating and managing SmbPndk resources.

Usage: smb [OPTIONS] <COMMAND>

Commands:
  login  Login to your account. To create an account, visit https://smbpndk.com
  help   Print this message or the help of the given subcommand(s)

Options:
  -l, --log-level <LOG_LEVEL>  Log level: trace, debug, info, warn, error, off
  -h, --help                   Print help
  -V, --version                Print version
```
