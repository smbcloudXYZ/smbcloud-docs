# smbcloud-docs

> [!IMPORTANT]
> **This repository is retired.** The smbCloud documentation site now lives in the
> [`smbcloud-cli`](https://github.com/smbcloudXYZ/smbcloud-cli) monorepo under
> [`website/`](https://github.com/smbcloudXYZ/smbcloud-cli/tree/main/website).
>
> It is deployed from there (`smb deploy -p docs`), and the developer docs are
> generated in place from the repo's own `docs/` tree — no separate sync. This
> repository is kept read-only for history; please open issues and PRs against
> `smbcloud-cli`.

The content here is the final pre-migration snapshot. It is byte-for-byte the same
as the version now maintained in `smbcloud-cli/website/`.
